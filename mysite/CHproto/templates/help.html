Universal Buttons

    Back: Click this button to go back to the previous page.
    Home Icon: Click this button to navigate back to the Course View page.
    Logout: Click this button to log the current user out of the Class Helper application. The currently logged in user’s name will be displayed underneath the logout button.
    Help: Click this button for help on the current page.

Login
    From the login page a user is able to enter a username and password if they have an existing account. If an account does not exist then they can create a new one.

    Username: Field where you enter an existing username.
    Password: Field where you enter the password, which is then checked with the username for login verification.
    Login: Verifies the username and password match before logging the user in. IF they do not match then an error will be given informing the user the login was not successful.
    Register: Click this button to create a new user account. You will be redirected to a new page.

Register
    From the register page you enter your information to create a new user account which will be used to track attendance for all your courses. You will have to enter courses after you have finished creating an account.

    First Name: Field where you enter your first name.
    Last Name: Field where you enter your last name.
    Username: Field where you enter your desired username.
    Password: Field where you enter a password which will then be used along with the username for login verification.
    Confirm Password: Field where you re-enter the password to verify that the password is correct.
    Email: Field where you enter your e-mail address.
    Register: Click this button to complete creation of an account.

Home/Course View
    The Course View page is the main source of navigation in the Class Helper application. From here a user is able to add a new course.
    Existing courses are located above the add course button. A user is able to edit an exist course or navigate to the Roster, Add Attendance and Graph pages for that course.

General Page Functionality

    Add Course: Click this button to add a new course. You will be redirected to the Add Course page.

Existing Course Functionality

    Roster: Click this button to navigate to the Roster page.
    Add Attendance: Click this button to navigate to the Add Attendance page.
    Graph: Click this button to navigate to the Graph page.
    Edit: Click this button to edit the existing course information.

Add New Course
    The Add New Course page is where you add a new course and students for that course. There are three main sections on this page; Add Course (top), Add Student (middle) and Current Student Roster (bottom).
    After entering a course and all student information click the finish button to save and complete the course creation.

Add Course

    Course Name: Field where you enter the name of the course.
    Course Section: Field where you enter the Section number of the course.
    Total course hours: Field where you enter the total number of hours for the course.

Add Student

    First Name: Field where you enter a student’s first name.
    Last Name: Field where you enter a student’s last name.
    ID Number: Field where you enter a student’s student ID number.
    Major[s]: Field where you enter a student’s major[s].
    Email: Field where you enter a student’s e-mail address.
    Add Photo: Click this button to add a photo of the student. After clicking the button the user will be prompted to add select a photo from search dialog.
    Create Student: Click this button to save and complete the creation of a student.
    Below the Add Student section is list of the current students added to the course. They are sorted alphabetically by last name.

Edit Course
    The Edit Course page is where you modify an existing course. You are also able to access the Edit Students page from here by clicking on the edit icon next to a specific student’s name.
    After entering a course click the finish button to save and complete the course modification.
    The total number of student in a specific course is displayed at the bottom of the page.

Edit Course

    Course Name: Field where you enter the name of the course.
    Course Section: Field where you enter the Section number of the course.
    Total course hours: Field where you enter the total number of hours for the course.

Edit Student

Select the information you want to edit and click the edit button next to it, this will enable you to edit that information. Once you are finished editing the student’s information click the “Confirm” button to save and complete adding a new student.

Graph
    The Graph page displays the attendance in a pie chart for either a student or a class. This data is graphable by week, month or semester.

Graph

    The timeframe a graph displays is indicated above the pie chart. The pie chart is color coded for easy recognition.
    • Green = Full
    • Yellow = Tardy
    • Red = Absent, Unexcused
    • Blue = Absent, Excused
    In the percentage is shown for each of the attendance marks inside the its corresponding mark. For example, if a student is absent, unexcused 5% of the time “5%” will appear in the Absent, Excused pie slice.
    The name of the course is displayed in the upper right corner of the workspace. The student’s name and email are displayed below the course name if the graph is displaying student information.
